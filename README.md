# Skhoron Vault

Локальный менеджер паролей. Часть экосистемы Skhoron.

## Architectural invariant: Zero Network

`AndroidManifest.xml` не запрашивает `INTERNET` permission. Без него ОС на
уровне ядра блокирует сокеты для UID этого приложения — это архитектурный
инвариант, а не настройка по умолчанию.

Следующий шаг для усиления: вынести `vault/` в отдельный Gradle-модуль с
изолированными зависимостями + CI-lint, проверяющий отсутствие сетевых
классов.

## Статус: v0.3

- **Крипто**: Argon2id (64MB/3 итерации) + XChaCha20-Poly1305 через
  BouncyCastle, чистый JVM без NDK
- **Разделение ключей**: два независимых подключа через HKDF —
  `KeyContext.SQLCIPHER` для шифрования файла БД и
  `KeyContext.FIELD_ENCRYPTION` для шифрования отдельных полей. Компрометация
  одного подключа не даёт доступа ко второму слою
- **Хранилище**: Room поверх SQLCipher
- **Разблокировка**: проверка мастер-пароля через попытку открыть
  SQLCipher-БД — неверный пароль бросает исключение
- **Panic wipe**: 10 неверных попыток подряд → необратимое удаление файла БД
  и локальных настроек
- **Auto-lock**: настраиваемый таймаут (1–30 минут, слайдер в Настройках)
- **FLAG_SECURE**: список паролей и генератор недоступны для скриншотов,
  screen recording и превью в панели "Недавние"
- **Автозаполнение**: `AutofillService`, датасет предлагается только при
  точном совпадении домена страницы с сохранённым (anti-phishing invariant)
- **Локальный бэкап**: экспорт/импорт (соль + зашифрованная БД) через SAF —
  без сервера
- **Генератор паролей**: SecureRandom, дефолт 64 символа (~419 бит)

## Roadmap / не реализовано

- Экран просмотра и копирования отдельной записи
- Диалог подтверждения сохранения в AutofillService (сейчас сохраняет сразу)
- Аудит слабых/повторяющихся паролей, TOTP/2FA, офлайн-проверка утечек

## Сборка

Требуется Android Studio (Koala или новее).

Gradle Wrapper не включён в репозиторий. Перед первым запуском CI или
сборкой через `./gradlew` из терминала выполни один раз:

```
gradle wrapper --gradle-version 8.7
```

Это создаст `gradlew`, `gradlew.bat` и `gradle/wrapper/gradle-wrapper.jar`.
Закоммить эти файлы вместе с остальным кодом.

Сборка APK:

1. `File → Open` — открыть папку `SkhoronVault/`
2. Дождаться `Gradle Sync`
3. `Build → Build Bundle(s) / APK(s) → Build APK(s)`
4. Готовый файл: `app/build/outputs/apk/debug/app-debug.apk`
5. Установить на устройство
6. Настройки → Система → Языки и ввод → Автозаполнение → выбрать
   Skhoron Vault

## CI

`.github/workflows/ci.yml` — сборка + lint + проверка architectural
invariant (отсутствие `INTERNET` permission и сетевых классов в исходниках).
Требует наличия Gradle Wrapper (см. выше).
